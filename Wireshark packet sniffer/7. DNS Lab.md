# DNS Lab

The objective of this exercise is to use Wireshark and tcpdump to determine if your newtork is being hacked. 

## Clear your DNS Cache

** Note: These next steps are to be executed in the Windows machine

1. Open Command Prompt and execute the following command to clear your DNS Cache.


```javascript
  ipconfig /flushdns
  ```

2. Check if the dns flush occurred correctly by executing:

```javascript
  ipconfig /displaydns
  ```
