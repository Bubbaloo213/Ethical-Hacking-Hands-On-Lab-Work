# HTML with Embedded Objects

When you open a web page, your browser fetches the HTML content by sending a GET request, which may also include links to images, styles (CSS), and scripts (JavaScript) hosted on different servers. After retrieving the HTML, the browser sends additional GET requests for these linked resources, often fetching multiple items simultaneously to speed up loading. By analyzing the timestamps of these requests in Wireshark, you can understand how efficiently the browser loads the page and which resources are retrieved at the same time, highlighting the process of loading a web page and its components from various servers.

## Capture Main Page Request

1) Clear previous capture and start Wireshark as new
2) Run

```javascript
    wget -p -k -E -nd -4 http://gaia.cs.umass.edu/wireshark-labs/HTTP-wireshark-file4.html
  ```
<br />

If you're unable to run the **wget**, run

```javascript
    curl -O http://gaia.cs.umass.edu/wireshark-labs/HTTP-wireshark-file4.html
  ```
