# Detecting an ARP Spoofing Attack

The objective of this exercise is to write a Python program to detect an ARP spoofing attack. First, we'll create our own ARP table using a dictionary. Then, we'll check if any incoming packet changes an entry in this table. If a packet changes the table, we will consider it to be a potential attack.
