# Detecting an ARP Spoofing Attack

The objective of this exercise is to write a Python program to detect an ARP spoofing attack. First, we'll create our own ARP table using a dictionary. Then, we'll check if any incoming packet changes an entry in this table. If a packet changes the table, we will consider it to be a potential attack.

## Install the Python Scapy packagepip3in whose function is to intercept and parse the packets that passs through our NIC.

### 2. Copy and paste the following Python program and save it as ***arpDetector.py***

  ```javascript
  from scapy.all import sniff
  IP_mAC_Map = {}

  def processPacket(packet):
	    src_IP = packet['ARP'].psrc
	    src_MAC = packet['Ether'].src
	    if src_MAC in IP_MAC_Map.keys():
		       if IP_MAC_Map[src_MAC] != src_IP :
			          try:
				               old_IP = IP_MAC_Map[src_MAC]
			          except:
				               old_IP = "unknown"
		              message = ("\n Possible ARP attack detected \n" + "It is possible that the machine with IP address \n" + str(old_IP) + "is pretending to be " + str(src_IP) +"\n ")
			          return message
		 else:
				IP_MAC_Map[src_MAC] = src_IP

  sniff(count=0, filter="arp", store=0, prn=processPacket)
  ```

### 3. Run the following command:

  **kali@kali:~$**
  ```javascript
  sudo python3 arpDetector.py
  ```

* Let the program run to detect potential ARP Spoofin Attack
